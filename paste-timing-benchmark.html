Paste timing benchmark

Measures the gap between the paste event and the subsequent input event — the same window PasteNinja's detection waits on. Useful for sanity-checking whether the 150ms fallback timeout has comfortable headroom.

PasteNinja's fallback timeout is 150ms. Measurements over that would have caused a false-positive injection in the old fixed-wait design.
