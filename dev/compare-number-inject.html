type=number injection comparison

Tests whether PasteNinja's pre-filter is necessary, or if we can just inject the raw clipboard text directly. Enter a string below (or pick a preset), click Run, and see what each path produces.

Presets:

Pre-filter + inject

Strip /[^\d.eE-]/g, bail if not Number.isFinite, inject the stripped value via the prototype setter.

Bare inject

No filtering. Pass the raw clipboard string straight to the prototype setter on the type=number field.

Native paste

After clicking Run, the raw string is copied to your clipboard. Then click the field and press Cmd-V to see what the browser's own paste handler produces.
